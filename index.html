Stonehearth's Modding Guide

Welcome to the Stonehearth modding guide!

This is still work in progress, so there are missing pages and not all the links work yet.

The modding guide is a "how-to" approach to modding, step by step. We've divided it in several groups, from more basic to more advanced. We'll also include explanations about the different types of scripts that we can find in the game.

We'll be updating the online version of the modding guide in the future to finish the missing content and fix the links.

Happy modding!
